# PREREQUISITES

* [Go](https://golang.org/)
* [tuco](https://github.com/mcandre/tuco)

## Recommended

* a UNIX-like environment (e.g. [WSL](https://learn.microsoft.com/en-us/windows/wsl/))

## Postinstall

Register output of `go env GOBIN` to `PATH` environment variable.
